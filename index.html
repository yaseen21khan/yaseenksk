View Details

Leveraging the JPEG Privacy & Security (PNS) international standard developed by DoubleBench, OnePhoto offers a robust solution for verifying photo authenticity and protecting image integrity.

Watch demo video

Download app apk

Key Features:

Panorama Application

An android application developed for automatic panorama stitching and embeddong JPEG Privacy & Security Signature for authenticity,
the app is build using C++, java, NDK and SDK

View Details

Our Android-based application tackles common issues in image stitching, including stitching failures and distortion. Developed with the Android NDK (Native Development Kit) and Java, the application ensures precise and efficient stitching of panoramic images.The app uses a gyro sensor to automatically capture images based on orientation changes, which automates the stitching process and enhances user experience. This automation streamlines the creation of panoramic images, reducing manual effort. Moreover, the application integrates advanced privacy and security features with a signature based on the JPEG Privacy and Security standard. This ensures the integrity and authenticity of the captured images. The images provided illustrate the application's capability to generate high-quality panoramic photos, demonstrating its effectiveness in producing clear and accurate stitched images.

Technologies: Android Gyro, OpenCV, NDK, SDK, C++ and Java

Download APK

Key Features:

  • Android-based application
  • Address the issue of stitching failure
  • Address the issue of distortion
  • Control events with an orientation sensor for automation

Publication: Automatic Sequential Stitching of High-Resolution Panorama for Android Devices Using Precapture Feature Detection and the Orientation Sensor

Vision Based Hand Gesture Recognition

This project leverages deep learning to enable control of drones using vision-based hand gesture recognition (HGR).

View Details

Initially, static hand gestures were classified using models trained on HGR datasets. For dynamic gestures, more advanced techniques such as LSTM, MediaPipe, and Transformers were employed for video-based hand gesture classification. These models achieved high accuracy, making it a robust system for real-time drone control through gesture commands.

Technologies: Python, Mediapipe, Tensorflow, Pandas, Numpy Arrays etc.

Watch demo video

Key Features:

  • Used MediaPipe, TensorFlow, and Python for model development.
  • The vision-based model was developed using state-of-the-art techniques to control the drone in real-time.
  • The system was successfully interfaced with a drone simulation in Unity.

Publication: Next-Gen Dynamic Hand Gesture Recognition: MediaPipe, Inception-v3 and LSTM-Based Enhanced Deep Learning Model
